## 🎯 核心特性

- **六维分类体系**：生存基线 / 精力与健康维护 / 杠杆与生产力 / 资产与储备 / 弹性消耗 / 资金流入

- **四层备份保障**：本地 + Git版本 + 飞书云 + GitHub远程

## 📝 记账格式

```markdown
- YYYY-MM-DD | [收入/支出] | [主分类] | #子标签 | ￥金额 | 原始完整输入
```

**示例：**
```markdown
- 2027-03-11 | 支出 | [生存基线] | #午饭 | ￥25.00 | 午饭25
- 2027-03-11 | 收入 | [资金流入] | #工资 | ￥5,000.00 | 3月工资
```

## 🔄 使用流程

### 日常记账
1. 打开当前年度文件（如 `2027.md`）
2. 找到当前月份区块
3. 在"当月流水明细"追加新记录
4. 更新"当月数据面板"统计

### 年度切换（每年1月1日执行）
```bash
# 1. 归档旧年度
python scripts/new_year.py 2028

# 2. 开始在新文件记账
# 编辑 2028.md
```

### 数据备份
```bash
# 手动备份
bash scripts/backup.sh

# 或提交到Git
git add .
git commit -m "记账更新: $(date +%Y-%m-%d)"
git push
```
